

Host Name:                      ชื่อของอุปกรณ์ในเครือข่าย ที่ใช้เพื่อระบุตัวตนของคอมพิวเตอร์หรืออุปกรณ์ในระบบเครือข่าย เช่น อินเทอร์เน็ตหรือเครือข่ายภายใน
OS Name:                        ชื่อของระบบปฏิบัติการ ที่ติดตั้งและใช้งานอยู่บนอุปกรณ์คอมพิวเตอร์ หรือเซิร์ฟเวอร์ เช่น Windows 11 Pro
OS Version:                      หมายเลขเวอร์ชันของระบบปฏิบัติการ ที่บ่งบอกถึงการอัปเดตหรือการแก้ไขที่เกิดขึ้นในระบบปฏิบัติการนั้นๆ โดยแต่ละเวอร์ชันจะมีฟีเจอร์ใหม่ๆ, การปรับปรุงประสิทธิภาพ, หรือการแก้ไขบั๊กต่างๆ
OS Manufacturer:               ผู้ผลิตหรือผู้พัฒนาระบบปฏิบัติการ ที่สร้างและดูแลระบบปฏิบัติการนั้นๆ โดยมักจะมีชื่อที่เป็นที่รู้จักและเป็นแบรนด์ที่ใช้กันทั่วโลก เช่น Microsoft, Apple, Google, หรือ Canonical
OS Configuration:               การตั้งค่าของระบบปฏิบัติการ ที่เกี่ยวข้องกับวิธีการทำงานและการปรับแต่งต่างๆ ภายในระบบ เช่น การตั้งค่าอุปกรณ์และโปรแกรมที่ติดตั้ง
OS Build Type:                   ประเภทของการสร้างระบบปฏิบัติการ เช่น Retail (เวอร์ชันเต็มที่จำหน่ายทั่วไป) หรือ OEM (เวอร์ชันที่ติดตั้งมาพร้อมกับเครื่องจากโรงงาน)   
Registered Owner:              ชื่อเจ้าของที่ลงทะเบียน เมื่อคุณติดตั้ง Windows, ระบบจะถามชื่อเจ้าของและแสดงเป็นข้อมูลนี้
Registered Organization:      องค์กรที่ลงทะเบียน หากคุณติดตั้ง Windows ภายในองค์กร ระบบจะถามชื่อองค์กรและแสดงเป็นข้อมูลนี้
Product ID:                        หมายเลขผลิตภัณฑ์ ใช้ในการระบุเวอร์ชันที่ได้รับการเปิดใช้งานของระบบปฏิบัติการ
Original Install Date:           วันที่ติดตั้งระบบครั้งแรก ระบุวันที่ที่ Windows ถูกติดตั้งหรือเริ่มใช้งานในเครื่องนี้
System Boot Time:               เวลาที่ระบบเริ่มทำงาน หรือเวลาที่เครื่องคอมพิวเตอร์บูตขึ้นตั้งแต่เริ่มเปิดเครื่อง
System Manufacturer:          ผู้ผลิตระบบ ชื่อของผู้ผลิตหรือแบรนด์ของคอมพิวเตอร์ เช่น HP, Dell, Lenovo
System Model:                    รุ่นของระบบ ระบุรุ่นของคอมพิวเตอร์ เช่น Dell XPS 15, HP Envy 17

10. System Type
System Type:                     ประเภทของระบบ เช่น 64-bit Operating System หรือ 32-bit Operating System
Processor(s):                      ประเภทและจำนวนของหน่วยประมวลผล (CPU) เช่น Intel Core i7 หรือ AMD Ryzen 5
BIOS Version:                     เวอร์ชันของ BIOS (Basic Input Output System) ที่ใช้ในการบูตเครื่องและควบคุมการตั้งค่าเบื้องต้นของฮาร์ดแวร์
Windows Directory:             ตำแหน่งของโฟลเดอร์ Windows โดยปกติจะอยู่ที่ C:\Windows
System Directory:                ตำแหน่งของโฟลเดอร์ระบบ เช่น C:\Windows\System32
Boot Device:                        อุปกรณ์ที่ใช้ในการบูตเครื่อง เช่น Hard Drive, USB Drive, CD/DVD Drive
System Locale:                   

  
ภูมิภาคของระบบ เช่น en-US (ภาษาอังกฤษ - สหรัฐอเมริกา) หรือ th-TH (ภาษาไทย - ประเทศไทย)
Input Locale:                       ภาษาของแป้นพิมพ์ เช่น English (United States) หรือ Thai (Thailand)
Time Zone:                         โซนเวลา ที่ตั้งของระบบ เช่น UTC+7:00 Bangkok, Hanoi, Jakarta
Total Physical Memory:         หน่วยความจำ RAM รวมทั้งหมด ในเครื่อง เช่น 8 GB หรือ 16 GB
Available Physical Memory:    หน่วยความจำที่สามารถใช้งานได้ หลังจากที่ระบบใช้ไปแล้ว เช่น 5.6 GB จาก 8 GB
Virtual Memory: Max Size:     ขนาดสูงสุดของหน่วยความจำเสมือน ที่ใช้เมื่อ RAM หมด เช่น 32 GB
Virtual Memory: Available:     หน่วยความจำเสมือนที่ยังสามารถใช้งานได้
Virtual Memory: In Use:        หน่วยความจำเสมือนที่กำลังใช้งานอยู่
Page File Location(s):            ตำแหน่งของไฟล์เพจ ซึ่งเป็นไฟล์ที่ระบบใช้เมื่อ RAM เต็มและต้องใช้พื้นที่ฮาร์ดดิสก์ช่วยเสมือนเป็น RAM เพิ่ม


Domain:                              โดเมน ที่เครื่องคอมพิวเตอร์เชื่อมต่ออยู่ (ในกรณีที่เครื่องอยู่ในเครือข่ายองค์กร)
Logon Server:                       เซิร์ฟเวอร์ที่ใช้ในการเข้าสู่ระบบ หากเครื่องเชื่อมต่อกับโดเมนจะมีชื่อของเซิร์ฟเวอร์ที่ทำหน้าที่ตรวจสอบข้อมูลการเข้าสู่ระบบ

Link-local IPv6 Address         ที่อยู่ IPv6 ที่ใช้ในการสื่อสารระหว่างอุปกรณ์ภายในเครือข่ายย่อยเดียวกัน (ไม่สามารถใช้ติดต่อระหว่างเครือข่ายที่แตกต่างกันได้)
IPv4 Address                        ที่อยู่ IP เวอร์ชัน 4 ซึ่งเป็นมาตรฐานที่ใช้กันมากที่สุดในโลก โดยที่อยู่ IPv4 จะประกอบไปด้วย 4 ชุดของตัวเลข ซึ่งแต่ละชุดมีค่า 0-255 และคั่นด้วยจุด (.)
Subnet Mask                          ตัวบ่งชี้ว่าเครือข่ายย่อย (subnet) มีขนาดเท่าใด โดยจะใช้บอกว่าที่อยู่ IP ของเครื่องนั้นอยู่ในเครือข่ายไหนและเครือข่ายย่อยไหนประกอบด้วย 4 ชุดของตัวเลข 0-255 เช่นเดียวกับ IPv4 Address โดยปกติจะใช้รูปแบบเช่น 255.255.255.0
Default Gateway                    ที่อยู่ IP ของอุปกรณ์ (เช่น เราเตอร์) ที่ใช้สำหรับการส่งข้อมูลออกไปยังเครือข่ายอื่น (เช่น อินเทอร์เน็ต)
